Control Systems: Control Network Interface
['Professional']
<p>Foxboro™ Distributed Control System (DCS) is a self-contained hardware and software solution for process control systems. Foxboro DCS includes Control Network Interface (CNI), which facilitates Ethernet communication between two Foxboro DCSs over a customer-supplied connection. CNI enables communication between various control processors across the plant.</p>

<p>This course provides essential knowledge on the basic hardware components of CNI and its baseplate. The course also covers how to install, diagnose, and maintain CNI in a control network.</p>
