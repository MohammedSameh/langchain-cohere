5004 Control Software Historian
['Essential']
This course introduces tools for accessing and engineering Historian and stored data. In this course, you gain a fundamental understanding of the role of Historian as a plant-wide data provider. You also gain a fundamental understanding of how to use Historian Client to configure reports and to report and analyze Historian data. Through hands-on practice, you modify, manage, and query the extensive database.<br><br>The course is ideal for personnel who retrieve data on process performance, plant performance, or plant operations.<br><br><strong>Learning Objectives<br></strong><br>After completing this course, you can:<br>• Operate Historian to:<br>&nbsp; &nbsp;- Insert, update, retrieve, and display archived and live plant data.&nbsp;<br>&nbsp; &nbsp;- Import and store older data offline.<br>&nbsp; &nbsp;- Configure advanced event detectors and event actions.<br>&nbsp; &nbsp;- Perform archiving operations.<br>• Set up Historian and access data with the SQL Query statement.<br>• Retrieve dates from Historian using Historian Client tools.&nbsp;<br>• Configure an instance to include events activated by both user-defined conditions and data import values.<br>• Describe data storage planning.<br><br><strong>Prerequisites<br></strong><br>• Familiarity with Microsoft Office® tools<br>• 5001 Control Software Configuration Essentials, 5001FV Control Software Configuration Essentials with FoxView, or equivalent knowledge
