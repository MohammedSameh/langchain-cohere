Troubleshooting EcoStruxure System Advisor for Process Control
[]
<p>This Virtual Instructor-Lead Training (VILT) provides essential knowledge on data flow from Application Workstations (AWs) to EcoStruxure™ System Advisor for Process Control. System Advisor for Process Control offers a management solution for system configuration, alarm management, change tracking, and documentation. This course covers troubleshooting common issues with data updates in System Advisor and a few other possible issues that may occur during the execution of System Advisor. The most common data update issues include modules and Control Processor (CP) database issues. The course proceeds to explain the procedure to maintain and upgrade system hardware components, architecture, data flow, and handshake for System Advisor with a Bridge connection.<br>
