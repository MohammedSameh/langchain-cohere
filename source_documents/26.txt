Processing Field Signals
['Essential']
<p>This course provides essential knowledge on continuous control loops and their configuration for processing signals from field devices. Topics include the process flow for signal conditioning and configuration of AIN and AOUT blocks for processing field signals. Knowledge of continuous control loops, their types, and configuration for signal processing enables you to fulfill plant requirements.</p>

<p>For optimal viewing, launch the courses in a Microsoft Internet Explorer® browser.</p>

<p> </p>

<p><strong>Learning Objectives:</strong><br>
<strong><strong>After completing this course, you can:&nbsp;</strong></strong><br>
- Explain how continuous control loops enable signal processing in Foxboro Evo™.<br>
- Configure an AIN block to process the input signals in a plant process.<br>
- Develop an AOUT block to condition the signals and relay the processed output.</p>
