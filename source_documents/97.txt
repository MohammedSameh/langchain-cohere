Foxboro System Administration for Microsoft Windows 10 and Server 2016 (Virtual)
['Professional']
This course provides essential knowledge for performing system administration tasks on Microsoft® Windows® 10 and Microsoft Windows Server® 2016 systems and best practices to follow when performing these tasks. The tasks include restoring the Windows operating system, licensing this operating system, creating a Commit installation folder using System Definition, deploying security for the system, installing software in different control systems, and setting up system backup and restore operations.&nbsp;<br><br>The course also covers troubleshooting common issues.<br><br><strong>Learning Objectives<br></strong><br>After completing this course, you can:&nbsp;<br>• Implement best practices for installing Microsoft Windows 10 and Microsoft Windows Server 2016 in Foxboro™ systems.&nbsp;<br>• Apply best practices for installing Endpoint Protection (EPP).<br>• Use System Definition (SysDef) to install Foxboro™ Distributed Control System (DCS) Control Core Services (CCS).<br>• Manage CCS installation.<br>• Modify the startup sequence of a workstation.<br>• Follow best practices for backing up and restoring a workstation.<br><br><strong>Prerequisites<br></strong><br>• Completion of the course activity Performing Platform and Control Software Upgrades in Foxboro DCSs<br>• Knowledge of Foxboro DCS configuration<br>• Knowledge of control networks
