2004 AIM*AT (Virtual)
['Essential']
This course introduces you to the programming steps for accessing and engineering the AIM*AT™ suite of software applications.&nbsp;<br><br>In this course, among other tasks, you practice configuring an AIM*AT server to retrieve data and report output into a Microsoft® Excel® spreadsheet or web-based application. Through hands-on lab exercises, you acquire up-to-date information on creating, managing, and querying this database.&nbsp;<br><br>The course is ideal for plant personnel who retrieve data on process performance, plant performance, or plant operations.<br><br><strong>Learning Objectives<br></strong><br>After completing this course, you can:<br>• Describe the purpose and functions of AIM*AT software applications.&nbsp;<br>• Operate the AIM*Historian application to monitor various process variables.<br>• Configure and utilize reduction groups.<br>• Perform AIM*Historian archiving operations.<br>• Access real-time and historical data in table and trend formats.<br>• Set up the AIM*ODBC driver and access data with the SQL Select statement.<br>• Access data using AIM*OPC and tools embedded in the AIM suite.<br>• Build and view HTML reports using AIM*Inform.<br>• Access and modify instances using AIM*Historian utilities.<br>• Configure an instance to include events activated by both user-defined conditions and MDE values.<br>• Configure and execute reports using the I/A Series™ Report package.<br><br><strong>Prerequisites<br></strong><br>2001v8 Foxboro Configuration Essentials or equivalent knowledge
