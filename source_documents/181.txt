Solución de Problemas de EcoStruxure System Advisor para el Control de Procesos (Virtual)
['Professional']
<p>Esta Formación Virtual Dirigida por un Instructor (VILT) proporciona conocimientos esenciales sobre el flujo de datos desde las Estaciones de Trabajo de Aplicación (AWs) a EcoStruxure™ System Advisor para el Control de Procesos. System Advisor for Process Control ofrece una solución de gestión para la configuración del sistema, la gestión de alarmas, el seguimiento de los cambios y la documentación. Este curso cubre la resolución de problemas comunes con las actualizaciones de datos en System Advisor y algunos otros posibles problemas que pueden ocurrir durante la ejecución de System Advisor. Los problemas más comunes de actualización de datos incluyen módulos y problemas con la base de datos del procesador de control (CP). El curso procede a explicar el procedimiento para mantener y actualizar los componentes de hardware del sistema, la arquitectura, el flujo de datos y el handshake para System Advisor con una conexión Bridge.<br><br>Este curso incluye 3 sesiones presenciales virtuales de 3 horas cada una y estas actividades obligatorias:<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; - Cuestionario previo al curso antes de la inscripción (15 minutos) <br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; - Trabajo posterior a la sesión antes de la siguiente (total de 9 horas):<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; o La sesión 1 requiere 1 hora y 40 minutos<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; o La sesión 2 requiere 4 horas y 20 minutos<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; o La sesión 3 requiere 3 horas<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; - Evaluación posterior al curso después de completar todas las sesiones (30 minutos) <br><br>Los participantes deben aprobar la autoevaluación previa al curso con una calificación de 75%.
