Getting Started with EcoStruxure Hybrid DCS
['Essential']
<p>This course provides essential knowledge on EcoStruxure™ Hybrid Distributed Control System (DCS).</p>

<p>Building complex control systems requires the coordinated efforts of multiple team members. EcoStruxure Hybrid DCS integrates control and supervision configurations with field devices in a single software environment.<br>
This course begins with clearing the misconceptions about EcoStruxure Hybrid DCS, highlighting its benefits. The course includes project scenarios in which EcoStruxure Hybrid DCS represents the most cost-benefit solution. The course concludes with EcoStruxure Hybrid DCS architecture, including software and hardware components.</p>

<p>For optimal viewing, launch the courses in a Microsoft Internet Explorer® browser.</p>

<p><strong>Target Audience</strong><br>
• Control Engineers<br>
• Maintenance Engineers&nbsp;<br>
• Process Operators</p>

<p><strong>Learning Objectives</strong><br>
After completing this course, you can:&nbsp;<br>
• Determine when to use EcoStruxure Hybrid DCS.<br>
• Explain the architecture of EcoStruxure Hybrid DCS, including software and hardware components.</p>

<p><strong>Prerequisites</strong><br>
Knowledge of:<br>
• Process control concepts<br>
• EcoStruxure Control Expert (formerly Unity Pro™ software) and Citect® Supervisory Control and Data Acquisition (SCADA)<br>
• Ethernet networking concepts<br>
• Modicon™ Programmable Logic Controllers (PLCs)</p>
