8903 TriStation 1131 Programación estándar (Virtual)
['Essential']
El curso de programación de nivel intermedio abarca técnicas avanzadas para escribir la lógica del programa a partir de un diagrama de flujo utilizando el lenguaje de diagrama de bloques de función (FBD). El curso también cubre la escritura de bloques de función del cliente utilizando tanto FBD como lenguajes de texto estructurado (ST).<br>Utilizando el software TriStation ™ 1131 Developer's Workbench compatible con EC61131-3, usted adquiere habilidades para el desarrollo efectivo de proyectos y la segmentación lógica comúnmente utilizada en aplicaciones de seguridad y control de procesos. Específicamente, usted desarrolla un proyecto, escribe y prueba la lógica del programa, particiona la lógica, asigna la memoria y descarga y programa.<br>Los ejercicios de laboratorio en tiempo real implican una declaración de diseño escrita, la segmentación lógica y la prueba del programa.<br>Este curso es ideal para el personal que programa o mantiene un sistema Tricon™, Trident™, o Tricon CX™.
Después de completar este curso, usted puede:<br>- Describir las operaciones del controlador lógico programable (PLC) TMR y Tricon™.<br>- Identificar los componentes del controlador Tricons.<br>- Explicar cómo reemplazar los módulos en línea en el sistema Tricon.<br>- Identificar y responder a las alarmas en el sistema Tricon.<br>- Utilizar el Monitor de Diagnóstico mejorado para:<br>&nbsp;&nbsp; o Conectarse al controlador Triconex.<br>&nbsp;&nbsp; o Supervisar el estado del sistema y solucionar los problemas del sistema.<br>&nbsp;&nbsp; o Identificar, informar y eliminar fallos.<br>&nbsp;&nbsp; o Recoger eventos.
