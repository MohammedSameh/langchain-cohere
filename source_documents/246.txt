2009 Advanced Foxboro System Administration (Virtual)
['Professional']
This course introduces you to command-based tools in a Foxboro™ system.&nbsp;<br><br>In this course, you use Foxboro tools to view communication and connectivity between stations. You view Object Manager connections, Inter-Process Communication (IPC) connections, and interactions between operator stations, Control Processors (CPs), and Historians. You also view CP usage and determine better control sizing.&nbsp;<br><br>Course activities concentrate on scripting capabilities, including writing scripts in Mortis Korn Shell (MKS) on windows-based systems.<br><br><strong>Learning Objectives<br></strong><br>After completing this course, you can:<br>• Explain MKS commands, filters, and scripting.<br>• View and report Object Manager and IPC connections.<br>• View CP usage and files.<br>• Describe the usage of Compound Summary Access.<br>• Explain features available in Device Monitor.<br>• Use tools available in FoxDraw™ software.<br>• List and review AIM*API utilities.<br><br><strong>Prerequisites<br></strong><br>• 2001v8 Foxboro Configuration Essentials<br>• 2007 System Administration for Foxboro Systems
