8602A Tri-GP / TriStation 1131 Configuration and Implementation Overview (Virtual)
['Essential']
This course covers Triconex™ General Purpose (Tri-GP™) implementation and maintenance. Specific topics include the principles of Tri-GP redundant architecture, system configuration, programming, maintenance, and troubleshooting. Using TriStation™ 1131 Developer’s Workbench software, the instructor demonstrates how to configure and program a system with digital and analog I/O. The course enables you to gain practical experience in system operation, including downloading new programs and enabling and disabling I/O points.<br><br>Learning modules target a detailed Tri-GP system overview, project development, online maintenance, and troubleshooting.<br><br><strong>Learning Objectives<br></strong><br>After completing this course, you can:&nbsp;<br>• Describe the basic theory of operation of Tri-GP architecture, including:<br>&nbsp; - Component identification<br>&nbsp; - Module replacement<br>• Develop a TriStation 1131 Developer’s Workbench project with Function Block Diagram (FBD) programming.<br>• Download procedures to the Tri-GP system.<br>• Perform online maintenance, including enabling and disabling points and download procedures.<br>• Perform system administration.<br><br><strong>Prerequisites<br></strong><br>• Working knowledge of Programmable Logic Controllers (PLCs) in a safety environment<br>• Familiarity with basic electronics and maintenance procedures
