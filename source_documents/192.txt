Essential Practices for Advanced Control Network Management (Virtual)
['Expert']
This course provides essential knowledge on effectively managing Foxboro™ DCS Control Network. Course topics include design, configuration, integration, partitioning, and troubleshooting. The course focuses on Switch Configuration Application Software (SCAS) and the Control Network Interface (CNI) module.&nbsp;<br><br>Course topics include network topologies, Rapid Spanning Tree Protocol (RSTP), switch interoperability, integration with device monitors, and recommendations for integrating and partitioning Foxboro systems.<br><br>This course is ideal for personnel who seek familiarity with concepts and techniques for managing advanced control networks&nbsp;<br><br><strong>Learning Objectives</strong><br><br>After completing this course, you can:&nbsp;<br>• Outline control network architecture for connecting control system components.&nbsp;<br>• Monitor, diagnose, and troubleshoot issues on the control network.<br>• Perform maintenance activities for device and data access on the control network.&nbsp;<br>• Configure network switches for transmitting control data using SCAS.&nbsp;<br>• Install, configure, and maintain CNI for integrating and partitioning networks.<br><br><strong>Prerequisites<br></strong><br>• Familiarity with control networks and networking concepts<br>• At least 1 year of experience in process control<br>• Experience installing and configuring Foxboro Control Core Services v9.x and Control Software v6.x<br>• Knowledge of Foxboro DCS hardware, including FCP280 and I/A hardware
