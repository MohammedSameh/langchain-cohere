Build and Manage a HART System Using Field Device Manager (Virtual)
['Professional']
This course provides essential knowledge on basic concepts of HART® protocol and instrumentation to help you manage HART instrumentation using Foxboro™ Field Device Manager for HART. In this course, you explore HART principles and main components. You also create and customize HART device templates. Lastly, you use basic maintenance and troubleshooting functions of Field Device Manager.<br><br><strong>Learning Objectives<br></strong><br>After completing this course, you can:&nbsp;<br>• Introduce basic concepts of HART instrumentation using Field Device Manager.<br>• Perform off-line steps to install new HART equipment.<br>• Identify HART equipment and install HART devices for use with Field Device Manager.<br>• Deploy and commission HART devices.<br>• Perform maintenance on HART devices using Field Device Manager for HART.<br>• Track changes after HART devices go online.<br><br><strong>Prerequisites<br></strong><br>• Familiarity with industrial plant instrumentation&nbsp;<br>• Knowledge of field device integration
