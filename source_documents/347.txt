8945A Triconex Safety Validator Overview (Virtual)
['Essential']
This course helps you improve plant availability and utilization by implementing the features of Triconex™ Safety Validator. The primary objectives of this course are to help you navigate the Safety Validator software and create and run a test project for validating a TriStation™ 1131 software application. The easy-to-use Safety Validator interface provides tools for running tests on Triconex Emulator and on Tricon™, Trident™, Triconex General Purpose (Tri-GP), and Tricon CX™ controllers.&nbsp;<br><br>This course is ideal for personnel who use Safety Validator to test and maintain online safety-critical applications<br><br><strong>Learning Objectives</strong><br><br>After completing this course, you can:&nbsp;<br>• Describe the basic features of Safety Validator.&nbsp;<br>• Open a project with TriStation 1131 software and download the project to Emulator.<br>• Open Report Generator to create a test project.<br>• Build a Test Project and create test suites, test cases, and test steps using Safety Validator.<br>• Create and edit templates.<br>• Run tests.<br>• Generate documents.<br><br><strong>Prerequisites<br></strong><br>• Working knowledge of Programmable Logic Controllers (PLCs) in a safety environment<br>• Familiarity with basic electronics and maintenance procedures
