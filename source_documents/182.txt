SAUTR1003 EcoStruxure Control Expert (Formerly Unity Pro Programming)
['Essential']
This course helps you acquire the knowledge to design, configure, install, connect, program, commission, and document M340 and M580 Programmable Logic Controller (PLC) projects using EcoStruxure™ Control Expert software.<br><br><strong>Learning Objectives&nbsp;</strong><br><br>After completing this course, you can:&nbsp;<br>• Identify the main features and benefits of EcoStruxure Control Expert software.<br>• Implement the five Control Expert programming languages.<br>• Distinguish the differences between Function Blocks and Derived Function Blocks.<br>• Compare the various processor hardware platforms supported by Control Expert.<br>• Demonstrate competency in implementing Program Organization Units (PUs).<br>• Manage variables with the Microsoft® Excel® tool.<br>• Explain the use of Field Device Tool (FDT) and Device Type Manager (DTM) technology in the software.<br>• Demonstrate confidence in the use of Change Configuration On The Fly (CCOTF).<br><br><strong>Specific Course Topics<br></strong><br>This course includes the following topics.<br><br><strong>Day / Session 1<br></strong><br>• EcoStruxure Control Expert at a Glance<br>• Controller Programming<br>• Simulating and Testing a Project<br>• Project Management<br>• Programming with:&nbsp;<br>&nbsp; -&nbsp; Ladder Diagram Language<br>&nbsp; -&nbsp; Function Block Diagram Language<br>&nbsp; -&nbsp; Function Blocks<br>&nbsp; -&nbsp; Structured Test Language&nbsp;<br>&nbsp; -&nbsp; Instruction List Language<br><br><strong>Day / Session 2<br></strong><br>• Graphics with Operator Screens Programming and Sequential Function Chart<br>• Communication to the PLC / PAC<br>• I/O Management<br>• Basic Diagnostic Tools<br><br><strong>Day / Session 3<br></strong><br>• Program Organization Units (PUs)<br>• Variable Management with Excel<br>• Variable References<br>• Embedded Data (Data Dictionary)<br>• FDT / DTM Technology<br>• Change Configuration On The Fly (CCOTF)<br>• Events and Maintenance<br><br><strong>Prerequisites<br></strong><br>Familiarity with:<br>• PLCs and Programmable Automation Controllers (PACs)<br>• Industrial automation<br>• Microsoft Windows operating systems<br>
