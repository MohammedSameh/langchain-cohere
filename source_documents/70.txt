5014 Foxboro Managed Endpoint Protection (Virtual)
['Expert']
This course helps you gain experience with Foxboro™ Secured Edition releases of McAfee® Endpoint Protection. Specifically, you gain an understanding of the secured configuration for a Foxboro system through the installation and setup of Managed McAfee ePolicy Orchestrator® (ePO™). Course topics address McAfee ePO installation, product navigation, configurable settings, and common tasks.&nbsp;<br><br>The course explores endpoint protection for potential threats and web events, rouge system detection, and whitelisting.&nbsp;<br><br><strong>Learning Objectives</strong><br><br>After completing this course, you can:&nbsp;<br>• Describe the installation methods of ePO.&nbsp;<br>• Define software used in ePO.<br>• View station information using ePO Console.<br>• Set up the Device Control feature.<br>• Create client tasks.<br>• Explain the technology of McAfee whitelisting and the Integrity Control feature of ePO.<br>• Describe, set up, and test Integrity Control.<br>• Build custom dashboards for viewing settings.<br><br><strong>Prerequisites<br></strong><br>• 5012 Secured System Administration or equivalent knowledge<br>• Minimum of 1 year of experience in process control administration
