Build and Manage a FOUNDATION Fieldbus System Using Field Device Manager (Virtual)
['Professional']
This course provides essential knowledge for configuring, commissioning, and maintaining FOUNDATION™ Fieldbus devices using Field Device Manager (FDM) in Foxboro™ DCS Control Software.<br><br><strong>Objectives</strong><br><br>After completing this course, you can:<br><br>• Explain the fundamental concepts of FOUNDATION™ Fieldbus technology.<br>• Describe the hardware and software requirements for implementing a FOUNDATION Fieldbus System in Foxboro™ Distributed Control System (DCS).<br>• Configure a FOUNDATION Fieldbus application using Foxboro Control Editors and Field Device Manager (FDM).<br>• Troubleshoot and maintain Fieldbus Module FBM228 and H1 devices using Foxboro System Manager and FDM.
