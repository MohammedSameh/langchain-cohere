Adding Endpoint Protection to Virtualized Control Systems
['Professional']
<p>This course guides you through using McAfee® &nbsp;ePolicy Orchestrator© (ePOTM) to add endpoint protection to the virtualized control system. This software helps you monitor and manage McAfee Management for Optimized Virtual Environment (MOVE) AntiVirus. The course also describes installing McAfee MOVE AntiVirus on a V91 server virtualization host.</p>

<p><strong>Target Audience:&nbsp;</strong><br>
• Application Engineers<br>
• Field Service Engineers</p>

<p>From Resources, you can download a job aid to help you perform procedures in this course.</p>

<p>For optimal viewing, launch the courses in a Microsoft Internet Explorer® browser</p>

<p><strong>Learning Objectives:</strong><br>
After completing this course, you can:&nbsp;<br>
• Import McAfee ePO Virtual Machine (VM).<br>
• Install McAfee MOVE AntiVirus.<br>
• Push virus definition database, known as DAT files from the McAfee ePO server to domain clients.</p>

<p><strong>Prerequisites:</strong><br>
Basic knowledge of:<br>
• Components of a virtualized control system<br>
• Concepts of cybersecurity</p>
